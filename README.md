# 顶格小程序

#### 介绍
微信博客小程序，完美兼容halo博客客户端。

#### 安装教程

1.  如果你已经有了基于halo的博客，那么恭喜你，可以直接更换域名，配置token即可运行且看到效果；
2.  如果没有，那么你可以前往halo的代码下载最新开发包本地安装，然后配置localhost:端口即可;
3.  xxxx
